# Lógica de Programação e Algoritmos com JavaScript

Este repositório contém a resolução dos exercícios e desafios propostos no livro **"Lógica de Programação e Algoritmos com JavaScript"** do autor **Edécio Fernando Iepsen** (Editora Novatec).

## 🎯 Objetivo

O foco principal deste repositório é o desenvolvimento do raciocínio lógico e a compreensão de algoritmos fundamentais utilizando JavaScript como linguagem base. O aprendizado abrange desde a manipulação de variáveis até a estruturação de dados em vetores e objetos.

## 📂 Organização do Repositório

A estrutura de pastas segue a ordem cronológica do livro para facilitar a consulta:

- **cap01-introducao:** Primeiros passos com saída de dados e operações básicas.
- **cap02-integracao-html:** Manipulação do DOM e eventos de interface.
- **cap03-condicoes:** Estruturas de controle (if/else, switch).
- **cap04-repeticao:** Laços de repetição (while, for).
- **cap05-vetores:** Manipulação de arrays e lógica de listas.

## 🛠️ Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript (ES6+)
